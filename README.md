# PowerLift Analytics

An end-to-end **Data Analytics** and **Machine Learning** project that analyzes powerlifting performance using the OpenPowerlifting dataset. The project tracks athlete progression, identifies performance trends, detects personal records (PRs), and predicts future lift performance using machine learning.

---

## Project Overview

PowerLift Analytics aims to transform raw competition data into meaningful insights for athletes, coaches, and analysts.

The project covers the complete data lifecycle:

- Data Cleaning
- Feature Engineering
- SQL Data Warehouse
- Exploratory Data Analysis (EDA)
- Interactive Dashboards
- Machine Learning
- Performance Prediction

---

## Objectives

- Analyze athlete career progression
- Track Personal Records (PRs)
- Measure lift improvements over time
- Identify performance plateaus
- Analyze competition trends
- Build a scalable SQL Data Warehouse
- Visualize athlete performance
- Predict future lift performance using Machine Learning

---

## Dataset

**Source:** OpenPowerlifting

https://www.openpowerlifting.org/

Dataset used:
- OpenPowerlifting Competition Results

Only **Indian athlete records** are used in this project.

> **Note:** The raw dataset is intentionally excluded from this repository because of its large size. The preprocessing scripts can be used to recreate the processed datasets.
